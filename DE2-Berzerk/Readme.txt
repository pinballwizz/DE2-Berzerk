Berzerk Arcade for the Altera DE2-35 Dev Board.

Notes:
Controls are PS2 keyboard, see readme file in de2 folder for instructions.

Build:
* Obtain correct roms file for berzerk, see make berzerk proms script in tools/berzerk_unzip folder for rom filenames.
* Unzip rom files to tools/berzerk_unzip folder.
* Run the make berzerk proms script in the tools/berzerk_unzip folder.
* Place the generated prom files inside the proms folder.
* Open the berzerk_de2 project file using Quartus and compile.
* Program DE2 Board.
